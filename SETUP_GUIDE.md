# Quick Setup Guide

## Step 1: Edit Configuration (2 minutes)

Open `lib/config/app_config.dart` and change:

```dart
static const String appName = 'Your App Name';
static const String targetUrl = 'https://your-website.com';
static const String appIconPath = 'assets/images/app_icon.png';
```

## Step 2: Add Your Icon

Place your app icon at `assets/images/app_icon.png`

## Step 3: Run

```bash
flutter pub get
flutter run
```

Done! 🎉
